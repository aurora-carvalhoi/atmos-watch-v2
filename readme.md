# 🌩️ ATMOS Monitor - Monitoramento de Infraestrutura

## 📌 Sobre o Projeto

O **ATMOS Monitor** é um sistema de monitoramento de infraestrutura voltado para servidores responsáveis pela distribuição de dados meteorológicos.

O projeto coleta métricas de desempenho das máquinas, processa esses dados por meio de um pipeline ETL e disponibiliza as informações de forma estruturada para análise e consumo por aplicações.

---

## 🎯 Objetivo

Garantir visibilidade e análise do desempenho de servidores através de:

- Coleta contínua de métricas de sistema
- Armazenamento centralizado de dados
- Processamento e padronização das informações
- Geração de dados prontos para visualização (time-series)
